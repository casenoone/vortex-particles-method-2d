# vortex-particles-method-2d
vortex particles for simulating smoke in 2d



## 背景简介
-涡粒子法 + 有限差分求解浮力场模拟烟雾扩散
-其中速度计算分别用了两种不同的势函数，comute_u_single与compute_u_single_other
-教程：https://zhuanlan.zhihu.com/p/425766028
